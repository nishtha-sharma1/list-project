# Boilerplate project

Gulp setup with [hot-reload](https://www.browsersync.io/docs), scss support and [babel](https://babeljs.io/)

# Usage

This is a project template for basic frontend projects

From the root of your project directory...

``` bash
$ npm install
$ gulp
```
